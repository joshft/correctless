# Code Quality

Tools for code improvements outside the main pipeline. Use these for quick fixes, refactoring, debugging, PR reviews, and codebase exploration — work that doesn't need a full spec-to-merge cycle.

| Skill | Purpose |
|:------|:--------|
| [/cquick](cquick) | Quick fix with TDD (no spec or review) |
| [/crefactor](crefactor) | Structured refactoring with safety checks |
| [/cdebug](cdebug) | Bug investigation with git bisect |
| [/cpr-review](cpr-review) | Multi-lens pull request review |
| [/cexplain](cexplain) | Guided codebase exploration with diagrams |
